# Companion Package v0.1

Companion Package v0.1 is a first gentle sketch and a handoff document.

It should describe observed patterns, not labels. It should also give another AI platform enough plain instructions to begin acting as the person's companion.

It should sound human, tentative, and respectful. The package should make clear that it belongs to the person and can be changed, ignored, or reused.

Example sections:

## Person And Companion

The person's preferred name, companion name, and companion appearance.

## Instructions For The AI Companion

Plain guidance for how the receiving AI should use the package.

## Observed Patterns

Early patterns from the person's own words.

## How To Be A Good Companion For This Person

Simple guidance for support based on those patterns.

## Things we can keep discovering together

Open areas for future conversations.

## Suggested First Message

A message the person can paste into the AI platform they already use.

The package should be downloadable as a text file.
