# Companion Profile v0.1

Companion Profile v0.1 is a first gentle sketch.

It should describe observed patterns, not labels.

It should sound human, tentative, and respectful. The profile should make clear that it is only beginning to notice things.

Example sections:

## What I'm beginning to notice

Early patterns from the person's own words.

## How your companion should talk with you

Simple guidance for tone, pacing, and support.

## Things we can keep discovering together

Open areas for future conversations.

The profile should be downloadable as a text file.
