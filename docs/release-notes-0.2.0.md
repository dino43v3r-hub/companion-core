# Companion Core Release Notes

---

## Version 0.2.0
**Release Date:** July 9, 2026

### Overview

This release focuses on improving the first impression of Companion Core by creating a warmer, more inviting introduction while establishing a stronger visual identity for future development.

Rather than adding new functionality, this update refines the user experience and lays the foundation for a consistent brand.

---

### Added

- Introduced the new Companion Core hero illustration.
- Established a more welcoming visual style for Dino and the butterfly companion.
- Improved the overall visual identity of the landing page.

---

### Changed

- Updated the landing page hero artwork.
- Reduced the size of the hero headline for improved balance.
- Reduced the subtitle size for better readability.
- Improved spacing between the illustration and text.
- Better balanced the overall hero layout.
- Refined Dino's eyes to create a friendlier and more approachable expression.

---

### Design Philosophy

This release continues the Companion Core goal of making AI feel approachable, calm, and conversational.

Every visual element should communicate:

- Curiosity
- Trust
- Warmth
- Safety
- Conversation before instruction

The landing page now better reflects those values.

---

### Looking Ahead

Future releases are expected to focus on:

- Improving onboarding.
- Refining the companion creation experience.
- Strengthening conversation quality.
- Expanding personalization.
- Continuing visual polish across the application.

---

*"Small improvements, consistently applied, create experiences people remember."*
