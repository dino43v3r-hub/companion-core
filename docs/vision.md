# Vision

Companion Core helps people begin shaping a digital companion that feels easier to talk with over time.

It is not a chatbot. It is a Companion Builder.

The first experience should feel calm, friendly, and human. The app should learn from how someone answers, not by putting them through a quiz.

The long-term idea is simple:

- Listen with care
- Notice patterns gently
- Help a companion become better suited to the person
- Keep the person in control

This MVP only builds the first onboarding moment.
